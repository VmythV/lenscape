# Lenscape

Lenscape, 中文名“镜界”，是一款 AI 拍摄指导 APP 的产品规划项目。

产品目标是帮助普通用户在拍摄前获得清晰、具体、可执行的姿势、站位、机位、角度和构图建议。镜界只提供拍摄前指导，不负责拍照、录像、上传成片、照片评分、自动修图或作品管理。

## MVP 定位

第一版聚焦“拍摄前指导”闭环：

```text
选择风格 -> 选择人物类型 -> 选择场景 -> 获取图文拍摄指导 -> 用户自行拍摄
```

MVP 建议优先实现：

1. 拍摄风格选择。
2. 人物类型选择。
3. 场景选择。
4. 姿势模板推荐。
5. 拍摄前文字步骤。
6. 姿势轮廓 / 边框可视化指导图。
7. 机位指导。
8. 构图指导。

## 实现判断

当前需求可以实现。第一版采用“结构化模板库 + 静态可视化模板图 + 规则推荐”为内容地基，并在其上叠加 LLM 做风格推荐与文案生成。

详细评估见 [docs/feasibility.md](docs/feasibility.md)。

## 技术形态（已确认）

| 维度 | 决定 |
| ---- | ---- |
| 客户端 | Flutter（iOS + Android 一套代码） |
| 内容数据 | 本地静态 JSON（风格 / 人物类型 / 场景 / 姿势模板配置打包进 APP） |
| 可视化指导图 | AI 文生图 + 人工审核的静态成图（边框 / 箭头 / 构图框画死在图上） |
| AI | 轻量后端代理，运行时调用 LLM 做①风格推荐 ②指导文案生成 |
| LLM 接入 | 统一抽象（工厂模式），同时支持 Anthropic 与 OpenAI 协议，可按配置切换 |
| 图片资产 | AI 文生图 + 人工审核 |
